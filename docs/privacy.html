Privacy Policy

Last updated: March 6, 2026

Overview

SplitView is a browser extension that splits your browser tabs into positioned windows. It collects zero user data. No analytics, no tracking, no telemetry, no network requests.

Data Collection

SplitView does not collect, store, transmit, or share any personal data or browsing information. Specifically:

  • No browsing history is accessed or recorded
  • No user identifiers are created or stored
  • No cookies are used
  • No analytics or tracking scripts are included
  • No data is sent to any server or third party
  • No remote code is loaded

Local Storage

SplitView stores minimal data locally on your device:

  • Your last chosen layout (e.g., "50-50" or "grid")
  • Temporary undo state (window positions) cleared on browser restart

This data:

  • Never leaves your device
  • Contains no personal information
  • Is automatically cleared when the extension is uninstalled

Permissions

SplitView requests only the permissions necessary for window management:

  • tabs — Required to query window and tab information for accurate positioning when splitting and restoring windows on undo
  • storage — Required to remember your last layout choice
  • system.display — Required to detect monitor dimensions for accurate window positioning on multi-monitor setups
  • contextMenus — Required to add "Open link in split view" and "Split this tab" to the right-click menu

Third-Party Services

SplitView does not integrate with, connect to, or communicate with any third-party services, APIs, or servers.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date. Since SplitView collects no data, meaningful changes are unlikely.

Contact

For questions about this privacy policy, please open an issue on the GitHub repository.
